Java八大字符类型

发表于2020-05-01|更新于2022-07-03|Java
|字数总计:258|阅读时长:1分钟|阅读量:

Java八大字符类型

*byte占一个字节范围：-128-127；
*short占两个字节范围：-32768-32767；
*int占4个字节范围；
*long占8字节范围
*float占4个字节
*double占8字节
*char占两字节
*boolean占一字节只有true和false两个；

1.整数型

byte numb1=15;
short numb2=15454;
int numb3=1546875484;
long numb4=4846498881454454L;//long类型要在数字后面加个L

2.浮点型（小数型）精度高

flout numb5=15.1f;//flout要在数字后面加个f;
double numb6=45.124;

3.字符型

char name1=’a’;
char name2=’李’;

4.字符串，string不是关键字，而是类；

string name3=”李霍霍”;

5.布尔型：只有是非

boolean flag=true;
boolean flag1=flase;

6.数组默认初始化值的规律（额外）

整数类型：默认初始值为 0

小数类型：默认初始值为 0.0

字符类型：默认初始值为 ‘/u0000’ 空格

布尔类型：默认初始值为 false

引用数据类型：默认初始值为 null（除了这四种之外的）
